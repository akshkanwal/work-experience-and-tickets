# IT Project Technician | F12.net

**Duration:** October 2021 – October 2023  
**Environment:** MSP — multi-client IT project delivery  
**Location:** Vancouver, BC (on-site and remote across BC)

## Role Overview

F12.net is one of Canada's largest MSPs, serving clients across multiple industries including healthcare, legal, financial services, and government. As an IT Project Technician, I was part of the project delivery team — executing on-site and remote IT deployments for F12's diverse client base.

This role expanded my technical range significantly: every client had a different environment, different standards, and different expectations. Working across multiple organizations simultaneously built adaptability, strong documentation habits, and the ability to quickly understand unfamiliar environments.

## Key Responsibilities

- Deployed and configured desktops, laptops, and peripheral equipment at client sites across BC
- Set up and networked printers and multi-function devices (MFDs) including driver deployment and print server configuration
- Performed hardware and software troubleshooting with root cause analysis documented in the ticketing system
- Provided end-user training and support both in-person and remotely via screen sharing tools
- Documented all installations, hardware configurations, and serial numbers in IT Glue and the ticketing platform
- Coordinated with project coordinators to ensure on-site work aligned with project scope and timeline

## Technologies Used

- Active Directory — user account management, group membership, OU management
- Enterprise imaging tools — WDS, MDT
- Ticketing systems — ConnectWise Manage
- Remote support tools — ScreenConnect, TeamViewer
- IT Glue — client documentation and asset management
- Printers and MFDs — HP, Canon, Ricoh, Konica Minolta
