# IT Infrastructure Specialist | F12.net

**Duration:** March 2024 – July 2024  
**Environment:** MSP — multi-client infrastructure design and delivery  
**Location:** Vancouver, BC

## Role Overview

The Infrastructure Specialist role at F12.net represented the highest technical level I held at the organization. Moving from project coordination back into a technical individual contributor role — but at a significantly higher level — I was responsible for designing and implementing LAN/WAN environments, server infrastructure, and security solutions for F12's enterprise clients.

This role brought together all of the networking, server, and security knowledge developed over the previous years and required applying it independently in complex, multi-site client environments.

## Key Responsibilities

- Designed and implemented LAN, WAN, VPN, and wireless network architectures for client environments
- Installed and configured network switches (Cisco, Meraki) including VLANs, spanning tree, port security
- Deployed and configured firewalls (WatchGuard, SonicWall) including policy design, NAT, and VPN setup
- Managed Active Directory, user accounts, OU structures, permissions, and Group Policy in multi-domain environments
- Implemented and maintained backup and disaster recovery solutions — backup policy design, testing, documentation
- Administered Microsoft Azure environments: VMs, virtual networks, Entra ID, Intune device management
- Authored technical documentation and SOPs for client environments in IT Glue
- Collaborated with the project team to deliver infrastructure projects on schedule and within scope

## Technologies Used

- **Networking:** WatchGuard, SonicWall, Cisco switches, Meraki, VLANs, VPN (IPSec, SSL)
- **Microsoft:** Azure, Intune, Entra ID, Active Directory, Windows Server 2019/2022, Group Policy
- **Monitoring & Management:** N-able N-central, IT Glue, PasswordState
- **Backup & DR:** Veeam, backup policy design and DR planning
