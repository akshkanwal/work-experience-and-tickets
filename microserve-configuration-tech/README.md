# Configuration Technician | Microserve

**Duration:** March 2021 – October 2021  
**Environment:** Enterprise device deployment and configuration  
**Location:** Vancouver, BC

## Role Overview

Microserve is a leading Canadian IT solutions provider specializing in enterprise device procurement, deployment, and managed services. As a Configuration Technician, I worked in their deployment centre preparing systems for delivery to enterprise clients across BC. This was my entry point into professional IT work, directly following the completion of my KPU CIS Diploma and CompTIA A+ certification.

The role required precision, speed, and consistency — systems had to be imaged, configured, tagged, and QA'd to exact client specifications before shipment. Any deviation from the configuration standard would affect the end-user experience across potentially hundreds of devices.

## Key Responsibilities

- Performed system imaging and OS configuration to client-defined standards using enterprise imaging tools
- Applied asset tags and maintained accurate inventory tracking in asset management spreadsheets
- Conducted QA checks on randomly selected systems from each batch before approval for shipment
- Configured peripherals (monitors, docking stations, keyboards, mice) and verified hardware functionality
- Escalated configuration issues to senior technicians with clear documentation of the fault and steps taken
- Collaborated with client IT departments to clarify configuration requirements and resolve discrepancies

## Technologies Used

- Enterprise imaging tools and WDS (Windows Deployment Services)
- Asset management and inventory tracking software
- Microsoft Excel for configuration tracking and QA logging
- Windows 10 — GPO-applied configurations, standard builds

## Notable Work

**Enterprise Clients Supported:**
- BC Health Authorities — medical-grade configuration standards, PHIPA-aligned builds
- ICBC — high-volume deployment with strict security and imaging requirements
