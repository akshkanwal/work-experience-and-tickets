# IT Consultant | ESW IT

**Duration:** April 2026 – Present  
**Environment:** MSP — Tier 2/3 consulting, project and ticket work  
**Location:** Vancouver, BC  
**Note:** ESW IT is currently in the process of being acquired by F12.net.

## Role Overview

ESW IT is a Vancouver-based MSP providing managed IT services and consulting to clients across BC. As an IT Consultant, I work at the Tier 2/3 level — handling complex escalated tickets, leading infrastructure projects, and providing technical advisory to clients on networking, security, Azure, and M365 decisions.

The acquisition by F12.net makes this a continuation of a long-term professional relationship, bringing the ESW client base into the F12 ecosystem while maintaining the same quality of service delivery.

## Key Responsibilities

- Resolve Tier 2 and Tier 3 project tickets and escalations across networking, Windows, Azure, and M365
- Administer and troubleshoot Azure infrastructure: VMs, virtual networks, resource groups, Azure AD / Entra ID
- Design and troubleshoot network environments including VLAN configuration and switch deployments
- Configure and troubleshoot SonicWall firewalls: security policies, NAT rules, VPN tunnels, IPS configuration
- Provide Windows environment support: AD administration, GPO troubleshooting, server issue resolution
- Scope and communicate technical solutions to clients — translating requirements into structured proposals
- Manage client calendar and scheduling for engineering and consulting engagements
- Collaborate with the sales team on technical recommendations and solution design

## Technologies Used

- **Cloud:** Microsoft Azure, Entra ID (Azure AD), Microsoft Intune
- **Microsoft 365:** Exchange Online, Teams, SharePoint, Conditional Access, MFA
- **Networking:** SonicWall firewalls, managed switches, VLANs, VPN
- **Windows:** Windows Server, Active Directory, Group Policy, DHCP, DNS
- **Remote Management:** Remote support tools, ticketing system
- **Security:** EPDR (endpoint detection and response), KnowBe4, MFA enforcement
